Software Update

MiraQ™ INTUI Update
 

The latest MiraQ release expands the functionality of the INTUI platform and builds on feedback from the initial launches. With a cleaner interface, improved compatibility, and smarter tools, this update reinforces MiraQ as the most intuitive, user-focused solution on the market.

Free Update

Existing INTUI Systems

Already running INTUI software? This update is available to you at no additional cost.

  

Paid Upgrade

Legacy Software (v4.3.2.2 or older)

Systems on version 4.3.2.2 or older can upgrade via the MiraQ INTUI System Upgrade Kit.

What's New

Key Highlights

Imaging PW Doppler Mode

The PW Doppler mode for ultrasound imaging has been re-introduced on the INTUI platform.

INTUI DICOM Option

INTUI now supports DICOM integration, with the added ability to export reports as PDF. Available as a field upgrade or option for new devices.

Toggle Color Coding in Reports

Users can now choose to remove color coding from flow values when generating a report.

Summary-Only Reports

Users can now generate a report containing only the summary page. Less wasted paper.

Chinese User Interface

The INTUI software now supports a full Chinese user interface.
